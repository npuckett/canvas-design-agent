How External Media Works in Canvas

Images: Instead of uploading files through Canvas, host them in a GitHub repository with GitHub Pages enabled. Reference them by URL in your HTML: https://username.github.io/repo/images/photo.jpg

Websites: Embed interactive content (p5.js sketches, data visualizations, custom widgets) via iframe. The hosted page can use JavaScript, CSS, SVG, and everything Canvas normally strips -- because it loads in its own document context.

Key advantage: Your media is version-controlled, shareable across multiple courses, and does not depend on Canvas file management. Update an image in git and it updates everywhere it is referenced.
