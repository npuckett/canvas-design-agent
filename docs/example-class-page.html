Lecture: Feedback Loops

Interactive systems need to respond to user input in meaningful ways. A feedback loop is the cycle of input → processing → output → perception that drives every interactive experience.

Key Concept: The three types of feedback loops in interaction design are positive feedback (amplifies action), negative feedback (corrects/dampens), and informational feedback (confirms/acknowledges).

Consider how the following everyday interactions use feedback:

  • A thermostat adjusting temperature (negative feedback)
  • Social media likes encouraging more posts (positive feedback)
  • A loading spinner confirming a process is running (informational feedback)

Discussion Prompt: Think about a frustrating interactive experience you've had recently. What kind of feedback was missing? How would you redesign the feedback loop?

Activity: Designing a Feedback Loop

In-Class Activity (50 min)

Work in pairs to design and prototype a feedback loop for one of the following scenarios:

  1. A plant watering system that communicates soil moisture
  2. A study timer that adapts to focus patterns
  3. A collaborative drawing tool with live presence indicators

Deliverable: A diagram showing input, processing, output, and perception stages. Identify which type of feedback loop you're using and why.

  • ☐ Choose a scenario
  • ☐ Map the input → output cycle
  • ☐ Identify feedback type (positive / negative / informational)
  • ☐ Sketch the interaction flow
  • ☐ Present to class (2 min per pair)
