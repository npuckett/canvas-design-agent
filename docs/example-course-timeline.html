Example: Course Timeline / Week Plan

A 14-week course schedule page using the Clean Modern (S01) theme with Forest Greens color palette swapped in.

About This Example

This page shows a course timeline built with the Clean Modern (S01) base theme, but with the accent colors swapped to the Forest Greens palette (#2d6a4f primary, #40916c secondary). This demonstrates how to start with a named theme and substitute a different color palette while keeping the same layout and typography.

Elements Used

  • S01 Clean Modern + Forest Greens
  • V05 Gradient Header
  • D05 Schedule Grid
  • V01 Left-Border Accent Box
  • V03 Alert / Callout Box
  • D07 Progress Bar
  • N01 Anchor Links
  • T01 Styled Headings
  • T02 Highlighted Text
  • C01 Collapsible Section

How to replicate: Give the agent your list of weeks, topics, and due dates as plain text, then tell it: “Use S01 Clean Modern but swap the accent palette for Forest Greens (#2d6a4f primary, #40916c secondary). Use a V05 header, D05 schedule grid, and highlight major deadlines with T02.”
