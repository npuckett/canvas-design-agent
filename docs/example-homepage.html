Example: Course Homepage

A welcoming landing page for a course, using the Warm Minimal (S03) theme with module navigation and key dates.

About This Example

This page demonstrates a course homepage built with the Warm Minimal (S03) theme. It uses earth-tone colors and the Palatino serif font to create an inviting, approachable feel suited to arts and humanities courses.

The homepage includes a welcome hero header, quick-link navigation, a module overview grid, key dates, and instructor contact information.

Elements Used

  • S03 Warm Minimal Theme
  • V05 Gradient Header
  • N02 Button-Styled Links
  • L03 Flexbox Row
  • C03 Info Card
  • V01 Left-Border Accent Box
  • T01 Styled Headings
  • D05 Schedule Grid
  • V03 Alert / Callout Box

How to replicate: Provide your course information as plain text and tell the agent: “Transform this into a Canvas course homepage. Use the Warm Minimal theme. Include a welcome message, module overview, quick links, and key dates.”
