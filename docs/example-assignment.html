Example: Assignment Page

A structured assignment page using the Editorial (S06) theme with Elegant Serif (F03) font stack for a typography-driven, grayscale look.

About This Example

This page demonstrates an assignment page built with the Editorial (S06) theme and the Elegant Serif (F03) font stack. This shows how to combine a named theme with a specific typeface. S06 is typography-driven and relies on grayscale contrast rather than color, while F03’s Palatino serif adds a refined, literary feel suited to writing-intensive courses.

The page includes assignment details, requirements, a collapsible rubric, submission instructions, and a late policy notice.

Elements Used

  • S06 Editorial Theme
  • F03 Elegant Serif Font
  • T01 Styled Headings
  • C09 Unordered List
  • C05 Definition List
  • C01 Collapsible Section
  • D01 Data Table
  • V03 Alert / Callout Box
  • V01 Left-Border Accent Box
  • N02 Button-Styled Link

How to replicate: Provide your assignment details as plain text and tell the agent: “Transform this into a Canvas assignment page. Use the Editorial theme (S06) with the Elegant Serif font stack (F03). Include requirements, rubric (collapsible), and submission details.”

Workshop Draft 2: Point of View Experiment

ENGL 204 • Module 2 • Due October 26, 2026

Write a short story (2,000–3,500 words) that experiments with an unconventional point of view. This draft builds on the voice exercises from Weeks 4–6 and pushes you to try a narrative perspective you haven’t used before.

Note: This is a workshop draft — it does not need to be polished. Focus on taking a genuine risk with point of view rather than producing a “finished” story.

Requirements

  • Length: 2,000–3,500 words
  • Must use a point of view other than simple past-tense third-person limited
  • Include a brief author’s note (100–200 words) explaining your POV choice
  • Double-spaced, 12pt font, standard manuscript format
  • File format: .docx or .pdf

Point of View Options to Consider

Second person (“you”)
See Lorrie Moore, “How to Become a Writer”
First person plural (“we”)
See Julie Otsuka, The Buddha in the Attic
Unreliable narrator
See Kazuo Ishiguro, The Remains of the Day
Shifting / multiple POV
See Jennifer Egan, A Visit from the Goon Squad
Unusual narrator (object, animal, collective)
See George Saunders, “Fox 8”
