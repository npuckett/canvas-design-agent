## Lecture: Feedback Loops (T01)

Interactive systems need to respond to user input in meaningful ways.
A **feedback loop** is the cycle of input → processing → output → perception
that drives every interactive experience.

### Key Concept (V01)

The three types of feedback loops in interaction design are
**positive feedback** (amplifies action), **negative feedback**
(corrects/dampens), and **informational feedback** (confirms/acknowledges).

Examples:
- A thermostat adjusting temperature (negative feedback)
- Social media likes encouraging more posts (positive feedback)
- A loading spinner confirming a process is running (informational feedback)

### Discussion Prompt (T06)

Think about a frustrating interactive experience you've had recently.
What kind of feedback was missing? How would you redesign the feedback loop?

## Activity: Designing a Feedback Loop (C03)

**In-Class Activity (50 min)**

B. A study timer that adapts to focus patterns

**Deliverable:** A diagram showing input, processing, output, and perception stages.
Identify which type of feedback loop you're using and why.
