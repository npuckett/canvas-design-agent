About

About this project and its author.

This tool was developed by Nick Puckett, Associate Professor at OCAD University.

Canvas Design Agent grew out of the practical challenge of creating well-designed, consistently formatted course pages in Canvas LMS without writing HTML by hand. It packages the knowledge of what Canvas will and will not render into a portable skill file that any AI agent can use to generate valid, visually polished page content from plain text.

This tool can be used, modified, or extended freely and openly for academic purposes.
