How It Works

01

Get the skill

Download SKILL.md and place it in your project or upload it to a web-based AI chat.

02

Write your content

Describe your Canvas page in plain text -- schedules, assignments, resources, anything.

03

Ask the agent

The AI generates Canvas-compatible HTML using only inline styles and tested elements.

04

Paste into Canvas

Switch to the HTML editor in Canvas RCE, paste, and save. Done.
